# 🗂️ Project YellowPages: Unified Search Directory

A premium, fast, and responsive client-side search website consolidating several business indices, local directories, and historical social archives. 

## 🚀 Features

* **Unified Search:** Real-time, instant fuzzy matching across names, categories, emails, phone numbers, addresses, and data sources.
* **Smart Category Filters:** Instant quick-filtering by major groups:
  * **X-Legacy Profiles:** Cleaned social handle data from the BickleRick 2020 Archive.
  * **Local Businesses:** Organic business pipeline list.
  * **Asset Management:** High-value corporate targets.
  * **Technology:** Industry-leading tech firms.
* **Premium Dark Mode:** Modern glassmorphic aesthetic built with clean CSS, tailored HSL color palettes, and interactive micro-animations.
* **Lazy Loading:** Dynamically renders chunks of 40 records to handle the consolidated database of 584 records seamlessly.
* **Click-to-Copy:** Quick copying of email, phone number, and unified contact strings.
* **Zero Dependencies / No Server Required:** Runs fully client-side. Simply open `index.html` in any browser.

---

## 📂 Consolidated Dataset

The directory merges, filters, and de-duplicates data from the following legacy sources:
1. **B-Raz Index Orgo:** AI-enriched core contact directory.
2. **B-Raz Index Sep22-25:** High-volume AI scraping of human technology frontrunners.
3. **GoodPipeline:** Years of organically collected local business listings.
4. **BickleRick legacy data:** Scraped, filtered, and cleaned X (Twitter) handles from 2020.

## 🛠️ Getting Started

### 1. Run Locally (Directly)
Since all data is packed in `data.js` and loaded dynamically via client-side JavaScript, you can run the app directly:
* Double-click **[index.html](file:///C:/Users/swank/.gemini/antigravity/scratch/yellowpages_search_project/Project-YellowPages/index.html)** to open it in your default browser.

### 2. Run Locally (via Dev Server)
If you prefer running it through a local HTTP server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx http-server -p 8000
```
Then visit `http://localhost:8000` in your browser.
